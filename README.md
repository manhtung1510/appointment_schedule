## Trước khi sử dụng

Cài đặt yarn cho máy

`yarn all` Nếu là máy thường

`yarn allM1` Nếu sử dụng chip M1

## Hướng dẫn sử dụng

#### `Trước khi chạy`

dùng lệnh `yarn start`

#### `Chạy IOS`

`yarn ios`

#### `Chạy Android`

`yarn android`
